# SudoDuel static site

This folder contains the tiny public SudoDuel website for App Store Connect.

## Deploy to Vercel

1. Import this repository into Vercel.
2. Set the project root to `site/`.
3. Framework preset: `Other`.
4. Build command: leave empty.
5. Output directory: leave empty.
6. Deploy.

## Public URLs to use after deployment

- Home: `https://sudoduel.app/`
- Support: `https://sudoduel.app/support`
- Privacy Policy: `https://sudoduel.app/privacy`

If you deploy to a temporary Vercel URL first, use the same paths on that hostname until the custom domain is connected.
